Featured Projects

Bioinformatics Chatbot

Developed the GENI-AI chatbot to guide researchers through Ma'ayan Lab web applications (e.g., Enrichr, ChEA3, Geneshot), enabling 10,000+ monthly users to analyze diverse data formats and receive figures and summary reports via natural language. Created 10 new MCP servers with the lab's applications and integrated with LangGraph's agentic framework, enabling the chatbot to autonomously chain multi-step bioinformatics workflows through dynamic tool composition and automated resource selection. Deployed GENI-AI and MCP servers on a 16-node Kubernetes cluster, implementing a caching system to efficiently store user data.

Multimodal Diffusion Models for Protein Generation

Developed zero-shot text-guided protein design pipeline using ProteinGenerator and ProtCLAP, generating novel proteins from text descriptions, achieving 97%+ structural confidence and 2x better alignment compared to unguided generation. Engineered a partial diffusion strategy (RFDiffusion) to explore protein dynamics, sampling 500 candidate structures of the Abl1 kinase domain and successfully mapping its functional conformational landscape by generating biologically relevant alternative states. Built using Python and PyTorch.
